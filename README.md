# mta-rss-feed
This is an MTA Service Change API made with assistance from Gemini and the official MTA API documents.

# How does it work?
This uses the URL of the official MTA Subway Service Changes API, basically runs it through some complete Python things (which I don't even understand at all), and, being from some complete .json text code as a whole that is complete nonsense, it makes it into some RSS feed to view! That is brilliant.
